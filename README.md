# Mai Trọng Thắng
B24DCKH117


1. Giải nén file ZIP ra một thư mục.
   (Lưu ý: CÓ Thể  chạy chương trình khi nó còn trong file ZIP)

2. Mở thư mục vừa giải nén.

3. Bấm đúp chuột vào file "app.exe" để chạy.

4. Một cửa sổ chọn ảnh sẽ hiện ra → chọn một ảnh bất kỳ (JPG, PNG, BMP…).

5. Chương trình sẽ tự hiển thị các kết quả xử lý ảnh.

NẾU BỊ LỖI
- Đảm bảo bạn đã giải nén đầy đủ.
- Đảm bảo file "app.exe" và các file .dll nằm chung một thư mục.
-Có thể thử lại bằng file ảnh KHÔNG nằm trong Onedrive
